# Run locally - VSCODE:

## Server:
1. open terminal in root folder
2. run ```cd server```
3. run ```npm i```
4. run ```chmod +x init.sh```
5. run ```./init.sh```
6. run ```node src/index.js```

## Client:
1. open terminal in root folder
2. run ```cd client```
3. run ```sudo npm i -g http-server```
4. run ```http-server -c-1```
5. go to http://localhost:8080
