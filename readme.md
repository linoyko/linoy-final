# Run locally - VSCODE:

## Server:
1. open terminal in root folder
2. run ```cd server```
3. run ```npm i```
4. run ```mkdir data```
5. run ```echo "[]" >> data/users.json```
6. run ```echo '"{}"' >> data/loggedInUser.json```
7. run ```node src/index.js```

## Client:
1. open terminal in root folder
2. run ```cd client```
3. run ```sudo npm i -g http-server```
4. run ```http-server -c-1```
5. go to http://localhost:8080
